# 什么是 O3O.ca ?

[O3O.ca](https://o3o.ca/) 是 [长毛象（Mastodon）](https://joinmastodon.org/)宇宙中的一个实例。

在这里，你可以通过发布“嘟嘟”来吐槽一切、摸鱼扯皮。无论你的兴趣点是什么，我们欢迎任何热情、友好、乐于分享的朋友们加入。请遵循互相尊重的原则，并遵守[《O3O主题或发言规则》](https://o3o.ca/terms)。
